### **_Description_**

Seafood Alert is a front-end application that utilizes NOAA's [FishWatch](https://www.fishwatch.gov/) API to search for and present information on US fisheries.

---

### **_Technologies Used_**

- Vue.js
- Surge
- Visual Studio Code
- Git/Github

---
